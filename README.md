### Example: 
  ```
  - movies/titanic return "Cameron" (movie director)
  ```
  ```
  - cinema/titanic return 2 (cinema room)
  ```
  ```
  - cinema/titanic/2 return "20:30" (movie time for that room)
  ```
  ```
  - review/titanic return 7.7 (movie rating)
  ```
